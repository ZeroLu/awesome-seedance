## 4. UGC 风格

此类别的重点是“用户生成内容”的美学，模拟日常拍摄的手机画面，并带有超现实的转折。

### 4.1. 超现实纪录片风格
*带有镜面反射转折的现实扭曲 Vlog。*

**提示词：**
```
【风格】伪纪录片（Vlog 风格），超写实，固定摄影机实拍感，自然光，带有一点悬疑喜剧色调。
【时长】15 秒
【主角】一个普通的年轻漂亮女性，在家中浴室水槽前。
[00:00-00:06] 镜头 1：日常设定（正常）。
场景：在普通的浴室镜子前。
动作：主角正在刷牙，满嘴泡沫。她对着镜子做各种滑稽的表情（眯眼、挑眉）。
关键细节：此时镜子里的倒影完全正常，动作同步。
[00:06-00:11] 镜头 2：出现 BUG（故障）。
动作：刷完牙后，主角低头吐出泡沫，然后转身离开浴室。
高能时刻（核心高潮）：就在主角的真实身体转身离开镜框时，镜子里的“倒影”**没有动**！那个“倒影”依然保持着刷牙的姿势，甚至坏笑着对着镜头挑眉，停留了整整 2 秒，才突然惊慌失措地“快进”赶上原主体的动作然后消失。
导演笔记：必须营造出极其真实的“网络延迟”感，仿佛倒影具有独立意识。
[00:11-00:15] 镜头 3：喜剧回扣（点睛之笔）。
动作：已经走到门口的主角似乎感觉不对劲，猛地回头看镜子。
结果：镜子现在已完全恢复正常，空无一人，只反射着对面的墙壁。主角困惑地挠了挠头，对着镜头露出怀疑人生的表情。画面定格在主角困惑的脸上（喜剧效果）。
